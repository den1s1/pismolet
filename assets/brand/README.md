# Письмолёт — графика для сайта

Папка содержит стартовый набор бренд-графики для лендинга и интерфейса проекта.

## Файлы

- `logo-horizontal.svg` — основной горизонтальный логотип для шапки сайта.
- `logo-stacked.svg` — компактная вертикальная версия для hero-блоков, карточек и заглушек.
- `logo-mark.svg` — знак без текста: конверт и бумажный самолёт.
- `logo-horizontal-mono.svg` — монохромная версия для нейтральных блоков и документов.
- `logo-horizontal-inverted.svg` — версия для тёмного фона.
- `favicon.svg` — favicon / компактная иконка сайта.

## Цвета

- Основной синий: `#2563EB`.
- Светлый синий: `#60A5FA`.
- Тёмный графит: `#1F2937`.
- Тёмный фон: `#0F172A`.
- Вторичный текст: `#64748B`.

## Идея знака

Знак объединяет конверт и бумажный самолёт: письмо не просто отправляется, а «летит». Это поддерживает название **Письмолёт** и позиционирование сервиса как простого способа отправлять email-рассылки без сложных настроек.

## Ограничения текущей версии

Файлы являются стартовыми SVG-ассетами для сайта и прототипа. Перед финальным публичным запуском стоит проверить:

- читаемость логотипа в маленьких размерах;
- отображение кириллицы в целевых браузерах;
- favicon на 16×16 и 32×32;
- юридическую чистоту названия и знака;
- необходимость ручной доработки типографики дизайнером.
