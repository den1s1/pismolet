Правила рассылок

Письмолёт предназначен для простых рассылок по своей законной базе. Сервис не предназначен для холодного спама, покупки баз и массовых отправок по чужим спискам.

1. Разрешённые сценарии

  • письма своим клиентам, участникам, подписчикам, волонтёрам или партнёрам;
  • информационные сообщения по базе, законность которой клиент может подтвердить;
  • рекламные сообщения только при наличии предварительного согласия адресатов;
  • рассылки после проверки дублей, очевидных ошибок и адресов, которые уже отписались через Письмолёт.

2. Запрещённые базы

Запрещены купленные, украденные, спарсенные, скачанные из открытых источников без согласия, чужие и иным образом незаконно полученные базы адресов.

Запрещено обходить ограничения сервиса: дробить рассылку на несколько аккаунтов, повторно загружать адреса, которые уже отписались через Письмолёт, или указывать недостоверный источник базы.

3. Подтверждения клиента

Перед отправкой клиент подтверждает источник базы и то, что у него есть законное основание для обработки адресов и отправки писем. Для рекламной рассылки клиент отдельно подтверждает наличие согласия на рекламу.

Сервис может запросить у клиента дополнительные пояснения или доказательства, если поступила жалоба адресата или рассылка выглядит рискованной.

4. Запрещённое содержание

Нельзя использовать сервис для незаконных предложений, вредоносных ссылок, обмана получателей, выдачи себя за другое лицо и иных нарушений закона или правил сервиса.

5. Отписка через Письмолёт

Если адресат отписался через Письмолёт, его адрес исключается из последующих рассылок сервиса независимо от клиента. Клиент не может отключить такую отписку или требовать отправки на такой адрес.

6. Модерация

Первые, крупные, рискованные или подозрительные рассылки могут проходить дополнительную проверку. Сервис может отказать в отправке без обязанности раскрывать все внутренние признаки проверки.

7. Последствия нарушения

При нарушении правил сервис может отказать в отправке, остановить рассылку, ограничить аккаунт, снизить лимиты или заблокировать дальнейшее использование сервиса.
