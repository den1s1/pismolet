Правила рассылок

Письмолёт предназначен для простых рассылок по своей законной базе. Сервис не предназначен для холодного спама и массовых отправок по чужим спискам.

1. Разрешённые сценарии

  • письма клиентам, участникам, подписчикам, волонтёрам или партнёрам;
  • информационные сообщения по своей базе;
  • рекламные сообщения при наличии отдельного согласия адресатов;
  • рассылки после проверки дублей, ошибок и глобальной отписки.

2. Запрещённые базы

Запрещены купленные, украденные, спарсенные, скачанные из открытых источников, чужие и иным образом незаконно полученные базы адресов.

3. Запрещённое содержание

Нельзя использовать сервис для мошенничества, вредоносных ссылок, экстремистских материалов, незаконных товаров и услуг, введения получателей в заблуждение и других нарушений закона.

4. Глобальная отписка

Если адресат отписался через Письмолёт, его адрес исключается из последующих рассылок сервиса. При попытке добавить такой адрес клиент должен получить предупреждение или исключение адреса из отправки.

5. Модерация

Первые, крупные, рискованные или подозрительные рассылки могут проходить дополнительную проверку. Сервис может отказать в отправке без обязанности раскрывать все признаки risk-score.

6. Последствия нарушения

При нарушении правил сервис может отказать в отправке, остановить рассылку, ограничить аккаунт или заблокировать дальнейшее использование сервиса.
