Письмолёт не для холодного спама

Сервис нужен для аккуратной коммуникации по своей законной базе: клиентам, участникам, подписчикам, волонтёрам, партнёрам и людям, с которыми у отправителя есть основание для связи.

Что запрещено

Эти правила нужны для безопасности получателей, клиентов и самого сервиса.

Купленные базы

Нельзя загружать базы, купленные у третьих лиц или скачанные из интернета.

Спарсенные адреса

Нельзя отправлять письма по адресам, собранным с сайтов, каталогов, соцсетей или утечек.

Чужие списки

Нельзя использовать базы, права и происхождение которых отправитель не может подтвердить.

Что проверяет сервис

  • клиент подтверждает законность базы при импорте;
  • для рекламной рассылки нужно отдельное подтверждение согласия на рекламу;
  • действует глобальная отписка на весь сервис;
  • первые и рискованные рассылки могут уйти на ручную проверку;
  • сервис может отказать в отправке при нарушении правил.

Есть сомнения по базе?

Лучше заранее описать источник адресов и сценарий рассылки. Это поможет понять, подходит ли Письмолёт.
