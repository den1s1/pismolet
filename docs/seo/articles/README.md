# SEO-статьи Письмолёта

В этой папке хранятся полноценные черновики SEO-статей для публичного сайта.

Редакционные правила:

- писать простым языком для малого бизнеса, НКО и локальных проектов;
- не создавать впечатление сервиса для нежелательных холодных отправок;
- явно повторять, что письма можно отправлять только по своей законной базе;
- не обещать юридическую безопасность или гарантированную доставку во входящие;
- вести читателя к посадочным страницам и CTA «Попробовать простую рассылку».
