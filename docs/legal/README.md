# Юридические документы Письмолёта

Статус: рабочий раздел юридической документации.
Дата создания: 2026-06-24.

## Актуальные документы

- `legal_evidence_layer_development_task.md` — задание на разработку юридического слоя доказательств: версии документов, подтверждения клиента, snapshot рассылок, жалобы, отписки и админка.
- `legal_texts_mvp_drafts.md` — черновики текстов для MVP: согласие клиента, поручение на обработку адресатов, декларация законности базы, подтверждение рекламного согласия, служебный блок письма, отписка, жалобы и запрос доказательств.
- `open_questions_decisions_2026-06-08.md` — ранее зафиксированные решения по открытым юридическим вопросам.

## Актуальное решение по базе адресатов

Письмолёт не собирает базы для заказчика и не реализует формы подписки адресатов в MVP.

Основной сценарий: клиент загружает свою базу email и подтверждает законность её происхождения. Double opt-in в MVP не используется.
