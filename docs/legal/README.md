# Юридические документы Письмолёта

Дата: 2026-06-24
Статус: рабочий раздел юридического чата

## Основные документы

- `app_ui_legal_texts.md` — юридические тексты для интерфейса приложения `app.pismolet.ru`: регистрация, профиль, импорт базы, подтверждения, рекламное согласие, оплата, запуск, глобальная отписка, жалобы и запрос доказательств.

## Важные решения

1. Основные формы регистрации, акцептов и чекбоксов живут внутри приложения `app.pismolet.ru`, а не на публичном сайте.
2. Письмолёт не собирает базы для клиента.
3. Письмолёт не реализует формы подписки адресатов и double opt-in в MVP.
4. Клиент сам загружает базу и подтверждает её законность.
5. Письмолёт технически обрабатывает данные и отправляет письма по поручению клиента.
6. Купленные, украденные, спарсенные, чужие и незаконные базы запрещены.
7. Рекламные рассылки требуют отдельного подтверждения клиентом наличия предварительного согласия адресатов.
8. Отписка действует глобально для всего сервиса.

## Что нужно синхронизировать

- Технический чат: реализовать чекбоксы, события логирования, версии текстов, `text_hash`, snapshot рассылки, глобальную отписку и workflow жалоб.
- SEO-чат: не использовать публичные формулировки, которые могут привлекать холодный спам или создавать впечатление, что можно загружать любую базу.
- Основная спецификация: раздел C должен ссылаться на тексты приложения как на источник интерфейсных юридических формулировок.
