## Что важно помнить при возврате к проекту

1. Админка теперь закрыта через allowlist. На сервере должен быть задан `Admin__AllowedEmails` или совместимый ключ конфигурации.
2. `Mailing.Recipients` содержит только адресатов, принятых к отправке. Исключённые строки проверяются через `LastImportStats` и `LastImportBatch.Issues`.
3. Тестовый запуск должен использовать `Testing`/InMemory/FakeMailer и не ходить в production Postgres.
4. После каждого следующего спринта обязательный gate:

```bash
dotnet build Pismolet.sln /nr:false -m:1
dotnet test Pismolet.sln --no-build
```
