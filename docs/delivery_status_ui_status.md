# Delivery status UI analytics

Дата: 22 июня 2026 г.

Статус: на проверке.

## Что добавлено

- В ЛК на странице запуска рассылки добавлена сводка по реальным статусам доставки.
- В ЛК добавлена таблица `Доставка по получателям`.
- В админской карточке кампании добавлен блок `Доставка`.
- В интерфейсе используются формулировки: `Доставлено`, `Временная ошибка`, `Постоянная ошибка`, `Отклонено`, `Не сообщено`.
- Диагностический текст почтового сервера показывается кратко, чтобы не перегружать таблицы.

## Что должно быть видно после деплоя

В ЛК:

- `Доставлено сейчас` в верхней сводке;
- `Доставлено`;
- `Временная ошибка`;
- `Постоянная ошибка`;
- `Отклонено`;
- `Не сообщено`;
- `Последнее событие доставки`;
- таблица `Доставка по получателям`.

В админке:

- блок `Доставка` перед логом отправки;
- счётчики по статусам доставки;
- таблица с email, статусом доставки, датой последнего события и причиной.
